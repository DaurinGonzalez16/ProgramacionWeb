Google Chrome, El cambio de los navegadores

Google Chrome es un navegador web de código cerrado.​​ desarrollado por Google, aunque derivado de proyectos de código abierto.​​​

Ventajas de Google Chrome

  1. Obtener resultados de búsqueda personalizados que aparecen a medida que introduces texto.
  2. Sincronizar marcadores y ajustes en todos tus dispositivos.
  3. Usar productos de Google Workspace, como Documentos, Hojas de cálculo o Presentaciones de Google.

Desventajas de Google Chrome

  • Sobrecarga en la memoria del dispositivo cuando hay muchas pestañas abiertas.
  • El navegador por si solo genera gran consumo de memoria RAM.
  • El almacenamiento de cookies puede resultar riesgoso para la privacidad del usuario.
